American Sign Language Classifier

An AI Program Designed to Identify ASL Hand Signs in Pictures

IMPORT IMAGE

Project Overview

Our model takes in photos of ASL hand signs as input and accurately identifies them with a letter label and bounding box.

  • Specifically, our model is capable of identifying the ASL signs for all 26 letters in the English alphabet, A - Z.

For Best Results

For our model to perform at its best, please follow the criteria below when uploading images:

  • Small, square image
  • Dim lighting (brightness not too high)
  • No sharp contrast
  • Plain background
  • JPG, JPEG or PNG file
  • Only one hand symbol per image

Applications

Our AI model can be used to:

  • Help people learn ASL
  • Ease communication with individuals who do not understand ASL

Resources

Kaggle

Kaggle is a website that has numerous datasets. We used ASL alphabet dataset for our project. The data set had 3900 photos of hand signs from A - Z

Labelbox

Labelbox is a tool we used ot label our dataset into the 26 letters from A-Z. With Labelbox, we were able to tell the computer which letter the image showed and build the training set.

Yolo

YOLO is a image detector that uses computer vision algorithms to identify objects in an image. We used YOLO to train our model to find and identify ASL signs.

AI Camp

AI Camp was where the team met and learned how to code Python under one week. From there, we started to build our ASL project.
